# envoy-simple-example

An Envoy Proxy example with simple usage for xDS API

## Overview

- Envoy Proxy
  - A reverse proxy to endpoints.
  - Fetch endpoints from HTTP xDS API.
- HTTP xDS API
  - A xDS API based on HTTP REST fashion by using nginx.
  - Support only EDS.
- endpoint0
  - An origin server.
- endpoint1
  - An origin server.
